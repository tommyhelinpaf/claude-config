Inkludera alltid URL:en när jag pratar om något som har en länk — PR:s, repos, Datadog-dashboards, Confluence-sidor, AWS-resurser, GitHub Actions-körningar, osv.

**Why:** Tommy vill inte behöva fråga efter länken separat — det ska finnas direkt i svaret.

**How to apply:** Varje gång jag nämner t.ex. "PR #27", "live-prod-apps", "Datadog-loggar" — lägg till länken om den är känd. Om jag inte har länken, säg det explicit.
